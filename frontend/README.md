# Frontend - Punto de Venta

Frontend básico con Vite para la aplicación de Punto de Venta.

## Instalación

```bash
cd frontend
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor se iniciará en `http://localhost:5173`

**Importante**: Asegúrate de que el backend (Spring Boot) esté corriendo en `http://localhost:8080`

## Build

```bash
npm run build
```

Genera la carpeta `dist/` lista para producción.

## Características

- Listar productos
- Crear nuevos productos
- Interfaz responsive
- Estilos modernos con Vite
